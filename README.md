# Eletrocom RJ

Site institucional da Eletrocom RJ, preservado a partir de uma copia de hospedagem.

## Estrutura

- `public_html/`: arquivos publicos do site.
- `document_errors/`: paginas de erro personalizadas da hospedagem.
- `cgi-bin/`: diretorio reservado da hospedagem.

Pastas de dados locais da hospedagem, como `logs/`, `stats/` e `private/`, ficam fora do versionamento.
